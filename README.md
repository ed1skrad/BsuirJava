# Генератор паролей

Это приложение, написанное с использованием следующих технологий:

- **Java 17**
- **Spring Web**
- **Spring Data Jpa**
- **PostgreSQL**
- **Spring Security**
- **AOP**

## Описание

Сервис принимает параметры сложности и длины пароля, а затем возвращает сгенерированный пароль.
